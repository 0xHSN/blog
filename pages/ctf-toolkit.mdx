## read pcap

```
sudo tcpdump -n -r password_cracking_filtered.pcap | awk -F" " '{print $3}' | sort | uniq -c | head
```

* `-n` skip DNS name lookups
* `-r` read from packet capture file
* `awk -F" " '{print $3}'` print the destination IP address and prot (the third space-separated field)
* `sort` sort it!
* `uniq -c` count the number of times the field appears in the capture
* `head` only display the first 10 lines of output
